# CloroPrime PWA Final — pacote corrigido com sw.js

## Arquivos principais
- `index.html`
- `manifest.json`
- `sw.js`
- `service-worker.js` (alias de compatibilidade)
- `icons/`

## Publicar no GitHub Pages
1. Crie um repositório.
2. Envie todos os arquivos desta pasta para a raiz do repositório.
3. Ative o GitHub Pages em **Settings > Pages**.
4. Abra o link publicado em **HTTPS**.

## Instalação no Android
- Abra no Chrome.
- Aguarde alguns segundos após a primeira carga.
- Use **Instalar app**.

## Instalação no iPhone/iPad
- Abra no Safari.
- Use **Compartilhar > Adicionar à Tela de Início**.

## Teste local no Windows
Execute `INICIAR_LOCAL_WINDOWS.bat`.
